## เกี่ยวกับโปรเจกต์

เป็นเว็บบริการจองห้องอ่านหนังสือ โดยมีการใช้ฐานข้อมูล users เพื่อให้ผู้ใช้กรอกข้อมูลเพื่อจอง <p>เเละระบบฐานข้อมูลจะมีการเก็บข้อมูลไว้
Front-end ใช้ Node js, React, Express
Back-end ใช้ phpmyadmin
